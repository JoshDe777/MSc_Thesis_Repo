### Abstract

This repository contains all project files for the project developed towards my Master's Dissertation at Trinity College Dublin. 

The `writing` folder contains all LaTeX source code and files for the written product of the project, while the `code` folder contains the Unity Project for the game developed. The Unity project is made using version `6000.4.8f1`, and would only change subject to a critical security update raised by Unity. 

This dissertation inspects the impact of game population (playing with/against CPUs vs real Humans) and crowd population (the filling of the stands in a stadium setting) on a player's sense of plausibility as defined by M. Slater as the impression/illusion that the happenings within a virtual environment are real. To that effect, I am developing a Beach Volleyball simulation for participants to play on, in order to conduct this operation in a controlled, digital environment.
